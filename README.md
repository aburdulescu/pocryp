## Install

### Precompiled binary

Go to [latest](https://github.com/aburdulescu/pocryp/releases/tag/v0.1.0)
release and download the tarball for your OS and CPU architecture.

### From source

If you have `go` installed, run:

```
go install bandr.me/p/pocryp@latest
```
